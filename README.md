Plataforma de governança do Curso de Habilitação ao Quadro Auxiliar de Oficiais
Bombeiro Militar — CBMRO. Uso interno da coordenação do curso.

## Onde fica cada coisa
- **Fonte + histórico (este repositório):** `~/Desktop/Site CHQAO` no Mac da Direção.
- **Pacote em uso / espelho da sala:** Google Drive da coordenação → `Meu Drive/SALAMANDRA`.
- **Backup externo:** este repositório privado no GitHub.
- **Banco de dados:** LOCAL em cada máquina (`~/.salamandra` no Mac, `%LOCALAPPDATA%\SALAMANDRA` no Windows) — nunca vai para a nuvem.

## Como salvar uma atualização
Edite aqui no repositório e rode **`Salvar-tudo.command`** (ou um `git commit`).
Um gatilho automático (`post-commit`) propaga para o **Drive** e para o **GitHub**.

## Como rodar
Dois cliques em `Iniciar-CHQAO.command` (Mac) ou `Iniciar-CHQAO.bat` (Windows).
Requer Node.js instalado.
